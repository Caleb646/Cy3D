# Cy3D: 3D Renderer using Vulkan and C++
Cy3D is a 3D rendering API written in C++17 that uses Vulkan. It currently only works on Windows. The plan is to wrap the API using Cython and make it accessible to Python3.
# Prerequisites
* Windows OS (Windows 10 or later).
* Visual Studio with C++ development tools installed.
* Vulkan SDK
# Additional Dependencies
* assimp
* glfw
* m3d
* shaderc
* spdlog
* spirv_cross
